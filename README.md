# **AZURO Creator**

**Discover physical and mathematical laws directly from your data with a click of a button!**

Simply upload a CSV file with input/output columns → click a button → get clear, interpretable formulas, accuracy (R²) and practical recommendations for the real world.

---

### **Main advantages**

* ✅ **100% local** – works offline, no cloud, no internet, no API keys
* ✅ **Interpretable formulas** – you see clear mathematical dependencies, no "black box"
* ✅ **Convenient visualization** – results are displayed in a clean and easy-to-read table
* ✅ **Calibration** – test known examples to check accuracy
* ✅ **Easy installation** – standalone Windows .exe, one file, no installation

---

### **How ​​it works (quick demo)**

1. Download the latest `.exe` version from Releases
2. Launch the application (double click)
3. Open in browser: [http://127.0.0.1:5000](http://127.0.0.1:5000)
4. Select an example or upload your own CSV file
5. Click **Discover Law** → see prediction table and top formulas
